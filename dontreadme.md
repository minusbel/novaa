# Nova Finance App

## Overview
Nova Finance is a modern, responsive financial web application built using React, Vite, and Tailwind CSS. The application provides users with features tailored for personal and business banking, loans, investments, and includes an interactive dashboard and live chat widget.

## Tech Stack
- **Framework**: React 19
- **Bundler**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Animation**: Motion (framer-motion)
- **Icons**: Lucide React
- **Forms**: React Hook Form
- **Language**: TypeScript

## Key Features
- **Authentication**: Login and user session management via AuthContext.
- **Client Portals**: Dedicated sections for Personal, Business, Loans, and Investments.
- **Dashboard**: A comprehensive, feature-rich user dashboard.
- **Support**: Integrated Live Chat Widget for customer support.
- **Resources**: A dedicated page for financial resources and learning.

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm (Node Package Manager)

### Installation
Run the following command to install the required dependencies:
```bash
npm install
```

### Development Server
To start the development server, run:
```bash
npm run dev
```

### Building for Production
To build the app for production:
```bash
npm run build
```

### Linting
To check for TypeScript/Lint errors:
```bash
npm run lint
```
